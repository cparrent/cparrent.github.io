# Law Enforcement Professional Portfolio Website

## Project Overview

The **Law Enforcement Professional Portfolio Website** is a professional web portfolio created as the final project for the Computing Applications course at Texas Tech University. The purpose of this project is to demonstrate the knowledge and skills gained throughout the course by combining project management principles with web development techniques.

This website highlights my experience, qualifications, certifications, and career goals in law enforcement while showcasing my ability to plan, develop, document, and deploy a complete website using HTML, CSS, GitHub, and GitHub Pages.

---

## Project Objectives

- Create a professional portfolio website using HTML and CSS.
- Demonstrate project planning using *Project Management for the Unofficial Project Manager (PMUPM)*.
- Organize project documentation using Markdown.
- Publish the completed project using GitHub Pages.
- Build a portfolio-quality GitHub repository.

---

## Website Features

- Professional homepage
- About Me page
- Responsive navigation menu
- Consistent styling using a single CSS stylesheet
- Professional color palette
- Organized project documentation
